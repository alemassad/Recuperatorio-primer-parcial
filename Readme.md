# Recuperatorio del primer parcial de programación para la administración
***
Este proyecto se desasrrolló con código HTML, CSS y JavaScript.
Cuenta con 1 página de Sing up para registrar un usuario.
La página HTML está linkeada a 2 archivos CSS y 1 achivo Javascript, separados en estructuras de carpeta estilos,para los archivos.ccs y carpeta script para el archivos javascript.js; además se ha sumado otra carpeta para la imagen de fondo.
Para resetear los estilos por defecto se implementó normalize.css. LA pantalla trata de contar con un
diseño agradable, tratando de respetar los estándares de UI/UX básicos y ser responsive para dispositivos moviles y pcs.
***
La página de Login contiene 6 input para ingresar el nombre, apellido, email, confirmacion del email contraseña, confirmación de la contraseña y además sumado a un botón  “Iniciar sesión” para realizar dicha acción dentro de un formulario.
Los inputs son llamados desde el archivo app.js para comprobar que los datos sean válidos.

***
Al precionar el botón registrar usuario, llama a un modal que indica al usuario que se ha registrado con éxito. Luego al cerrar el modal con un botón, el formulario se resetea limpiando todos los contenidos usados anteriormente.
